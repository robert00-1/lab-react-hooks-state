# Title
Shopping App

# Description
A shooping App is a simple react application demonstrating state management with react hook .
Users can toggle dark mode, filter products by category, and add item to  a shopping cart.

# Tools
Vite
React
CSS
React Hooks
Jest
Node.js/npm

# Setup
Follow this step to access and run the project locallt
clone the repository : clone lab-react-hooks-state in your local machine
git clone https://github.com/robert00-1/lab-react-hooks-state.git
- navigate into the project folder
 run : cd lab-react-hooks-state
 - install dependencies
  npm install
  npm run dev

  - copy paste the link that will appear after running npm run dev  https: //localhost:5173/  and run it on your browser
  you will see the Shopping App
